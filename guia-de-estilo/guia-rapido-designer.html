Guía rápida de estilo · SistemaNow
Reglas base para mantener el mismo diseño en todas las pantallas.

Usa estas referencias como “manual” para tarjetas, botones, estados y tipografía. El objetivo es que todo el sistema se vea coherente, limpio y estilo iOS / Apple.

Radio de las tarjetas
rounded-2xl ≈ 16–18 px
Color primario
#007AFF (botones, focos)
Fondo de tarjetas
bg-slate-900/70 + backdrop-blur-2xl
Avatar técnicos
JC
Fondo: #FF611A
1. Tarjetas y bordes
  • Todas las tarjetas principales usan rounded-2xl.
  • Bordes sutiles: border border-white/15.
  • Fondo “cristal”: bg-slate-900/70 o bg-slate-900/80.
  • Efecto vidrio: backdrop-blur-2xl (≈ 18–20 px de blur).
  • Sombras suaves sólo en tarjetas, no en botones.
2. Colores y tipografía
  • Fondo base: #020617 (muy oscuro, estilo panel Apple).
  • Texto principal: text-white / text-white/90.
  • Texto secundario: text-white/60 o text-white/70.
  • Tipografía: system-ui, -apple-system, "SF Pro Text".
  • Colores de estado siempre con transparencias: /20, /25, /30.
3. Botones y estados
  • Botón primario: bg-[#007AFF] + hover:bg-[#0060d6].
  • Botones secundarios: bg-white/10 + border border-white/20.
  • Estados de servicio y técnico en chips redondos: rounded-full.
  • Siempre usar “bolita” de color (dot) a la izquierda del texto para estados.
  • Títulos pequeños en mayúsculas: text-[11px] + tracking-[0.18em].
Siempre que se cree una pantalla nueva, seguir estas reglas: misma tipografía, mismos radios, mismos colores y misma lógica de estados. Así todo el sistema se siente como una sola aplicación, coherente y profesional.
